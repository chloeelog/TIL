## 접근방법

### 문제 살펴보기

- ~~조합으로 풀어야 할까? 부분집합에 더 가까운 것 같다!~~ 아니야! 경우의 수야! 해시맵으로 풀면 돼!
- 각 카테고리 내에서는 한 가지만 선택할 수 있다 / 모든 카테고리를 반드시 포함할 필요 없다 / 단, 한 가지는 반드시 입어야 한다

    ⇒ 즉, 어떤 카테고리에 n개의 원소가 주어져있다면, 해당 카테고리에서 아무것도 선택하지 않는 것을 포함하여 n+1가지의 경우가 나온다.

    ⇒ 각 카테고리별로 의상들을 선택하는 경우는 동시에 일어나므로, 각 카테고리별 경우의 수를 모두 구한 다음 서로 곱한다.

    ⇒ 모든 카테고리에서 아무것도 선택하지 않는 경우를 제외한다.

### 코드 작성하기

1. 해시맵을 생성한다: 키는 카테고리, 밸류는 카테고리에 해당하는 의상의 가짓수가 된다.
2. 주어진 배열을 순회하며 해시맵에 매핑한다.
3. 해시맵의 원소들을 순회하며 경우의 수를 곱한다.
    - reduce를 이용해 구해보자! ⇒ 불가능! 해시맵은 reduce를 쓸 수 없다.
4. 전체 경우에서 1을 뺀 값을 답으로 반환한다.

## 제출 코드

### 해시맵 이용

```jsx
function solution(clothes) {
    var answer = 1;
    
    // step 1. 해시맵을 생성한다. 
    let sH = new Map;
    
    // step 2. 주어진 의상을 해시맵에 매핑한다.
    for (let [value, key] of clothes) {
        sH.set(key, sH.get(key) + 1 || 1);
    }
    
    // step 3. 해시맵의 원소들을 순회하며 경우의 수를 곱한다.
    for (let [key, value] of sH) {
        // console.log(`${key}는 ${value}개 있다!`)
        answer *= (value + 1);
    }
    
    // step 4. 전체 경우에서 1을 뺀 값을 정답으로 출력한다.
    answer -= 1;
    
    return answer;
}
```
