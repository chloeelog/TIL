# HTML과 웹 브라우저

## HTML

> HyperText Markup Language

- `HyperText` :

  ```
  [명사] 사용자에게 비순차적 검색이 가능하도록 제공되는 텍스트. 문서 내의 특정한 단어가 다른 단어나 데이터베이스와 링크되어 있어 사용자가 관련 문서를 넘나들며 원하는 정보를 얻을 수 있다. (출처: Oxford Languages)
  ```

  - 사전적으로는 다른 데이터로 연결된 링크를 의미하나, 웹 문서를 구성하고 있는 요소들을 하이퍼텍스트로 볼 수 있음.

- `Markup`

  ```
  태그 등을 이용하여 문서나 데이터의 구조를 명기하는 언어. 
  ```

  - 여타의 프로그래밍 언어가 요소(데이터)를 조작하는 것과 달리, 마크업 언어는 요소의 구조와 역할을 정의하고 보여주되 조작하지는 않는다.

# Algorithm - 완주하지 못한 선수

# 접근방법

- 분류가 해시니까 해시키를 써볼까?

  - participant 배열을 돌며 해시맵에 맵핑하며 더하고

  - completion 배열을 돌며 해시맵에서 읽어들이면서 제외하자!

    - 이 때 밸류가 0이 된 해시키는 삭제해준다!

  - 남은 해시맵의 키를 리턴한다!

    ⇒ 해시맵 키 찾아내는 방법 뭘까? 왜 한번에 깔끔하게 안될까?

# 제출 코드

### 제출 코드 : 정답!

```jsx
function solution(participant, completion) {
  var answer = "";

  // 참가자 배열을 돌며 해시맵에 매핑한다.
  let sH = new Map();
  for (let p of participant) {
    sH.set(p, sH.get(p) + 1 || 1);
  }

  // 완주자 배열을 돌며 해시맵에서 키를 찾아 지워준다.
  for (let c of completion) {
    if (sH.get(c) === 1) {
      sH.delete(c);
    } else {
      sH.set(c, sH.get(c) - 1);
    }
  }

  for (let [key, value] of sH) {
    answer = key;
  }
  return answer;
}
```
