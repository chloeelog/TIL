## 접근방식

- 만약 논문 개수가 최저 인용 수 보다 적다면, 논문 개수 반환
- 그렇지 않은 경우, 역순으로 정렬해서
- index 가 인용 수 보다 커지는 경우, 직전 index 반환

## 풀이코드

```jsx
function solution(citations) {

  if (Math.min(...citations) >= citations.length) {
    return citations.length
  }

  citations.sort(function(a, b) {return b-a})

  for (let i=0; i < citations.length; i++) {
    if (i+1 > citations[i]) {
      return i
    }
  }
}
```



## 접근방식

```
앞 뒤 번호에게만 여벌 옷을 줄 수 있기 때문에

순서대로 옷이 없는 학생들의 앞 뒤를 체크해 옷이 있는 경우를

전체에서 옷이 없는 학생을 뺀 값에 더해 return했다.
```

## 풀이코드

```jsx
function solution(n, lost, reserve) {
  let student = new Array(n + 1).fill(1);
  for (let r of reserve) student[r]++;
  for (let l of lost) student[l]--;
  const dc = [-1, 1];
  let answer = n - lost.length;
  lost.sort(function (a, b) {
    return a - b;
  });
  for (let l of lost) {
    if (student[l] == 1) {
      answer++;
      continue;
    }
    for (let d = 0; d < 2; d++) {
      const next = l + dc[d];
      if (next == 0 || next == n + 1) continue;
      if (student[next] == 2) {
        student[next]--;
        answer++;
        break;
      }
    }
  }
  return answer;
}
```
